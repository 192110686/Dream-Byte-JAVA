Access Modifiers:
------------------
Access modifiers decide who can access a variable, method, class.

They are used for :
-> control
-> security
-> restricting unnecessary acess

public   -> can be accessed from anywhere 
private  -> can be accessed only inside the same class.
default  -> if no modifier is written, it can be accessed only inside the same package.
protected-> can be accessed in the same package and also in the subclass
