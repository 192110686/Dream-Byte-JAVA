List:
------------ 
List is an interface in java that stores elements in an ordered way and allows duplicate values.

List = ordered + duplicates allowed + index-based
eg:
-----
playlist
to-do list
shopping cart

Main features of List:
------------------
Feature 1: Insertion order is maintained.
A
B
C
o/p -> [A, B, C]
This is called maintaining insertion order.

Feature 2: Duplicates are allowed
eg:
---
[10, 20, 10, 30]

Feature 3: Index-Based access is possible.
Each element has a position number called index.
1st -> 0
2nd -> 1
So we can access elements by index.

Feature 4: Dynamic Size
List can grow or shrink dynamically.
This is unlike arrays, which are fixed size.

Feature 5: Stores Objects
List stores Objects, not primitive types directly.

Is List a class or Interface?
--------------------
List is an interface, not a class.
That means List gives rules, but actual objects are created using implementing classes.

->ArrayList
->LinkedList
->Vector

12. Traversing a List:
--------------------------
we have 2 options to traverse a list
1. using a normal for loop

import java.util.ArrayList;
import java.util.List;

public class ListSyntax {
    public static void main(String[] args){
  List<Integer> list = new ArrayList<>();
  list.add(1);
  list.add(3);
  list.add(1);
  list.add(4);
  list.add(1);
  System.out.println(list.lastIndexOf(1));
  for(int i=0;i<list.size();i++){
    System.out.print(list.get(i));
  }
    }
}

2. using enhanced for loop

import java.util.ArrayList;
import java.util.List;

public class ListSyntax {
    public static void main(String[] args){
  List<Integer> list = new ArrayList<>();
  list.add(1);
  list.add(3);
  list.add(1);
  list.add(4);
  list.add(1);
  System.out.println(list.lastIndexOf(1));
for(Integer li:list){
    System.out.println(li);
}
    }
}
