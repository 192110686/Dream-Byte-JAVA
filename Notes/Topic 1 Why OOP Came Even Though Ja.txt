Why Java Uses OOP:
------------------
Java is called as an Object-Oriented Programming language because java mainly follows OOP concepts to build applications.
In Java:

1. we create classes
2. from classes we create objects
3. inside them we keep variables and methods 

That is why OOP is the "heart of Java".

Main reasons OOP Came:
---------------------------
1. Basic programming becomes messy for big applications.
2. Too many variables and functions become hard to manage 
3. Real-world things need real-world modeling
4. Data and actions should stay together
5. code should be reusable 
6. code should be easy to maintain and update 
7. Large programs need better structure.
